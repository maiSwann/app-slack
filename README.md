# Slack

**Own version of slack on mobile**

Using :
------

- Alamofire
- API requests
- MVC
- Cocoapods
- https://github.com/John-Lluch/SWRevealViewController

Features :
---------
- Login and create account page
- Add a channel
- Send and receive messages in differents channels
- Real time update
- Notifications
